# S-Processor School Management System – Training Manual

---

## Purpose of This Manual
This training manual is designed for **teachers, administrators, and school staff** to quickly learn how to use S-Processor in daily school operations.

---

## Training Plan Overview

### Day 1: Basics
- Understanding the dashboard
- Navigating modules
- Setting school information

### Day 2: Student Management
- Registering students
- Editing student data
- Deleting students (with caution)

### Day 3: Exams & Results
- Entering marks (per class & subject)
- Generating reports
- Understanding positions

### Day 4: AI Remarks
- Generating remarks
- Editing and correcting remarks

### Day 5: Accounting
- Recording fees
- Tracking payments
- Identifying unpaid students

### Day 6: Reports & Performance
- Aggregation board
- Performance tracking

### Day 7: Backup & Safety
- Saving records
- Restoring backups
- Avoiding data loss

---

## Module Training Details

### 1. Dashboard Navigation
- Open the system
- Identify key sections:
  - Accounting
  - School Examination
  - User Tools

Exercise:
- Navigate between all modules

---

### 2. Student Registration

Steps:
1. Go to **User Tools → Register Student**
2. Select class
3. Enter student details
4. Click Register

Exercise:
- Add 3 students

---

### 3. Entering Exam Marks

Steps:
1. School Examination → Make School Exams → Per Class
2. Select class
3. Enter marks
4. Press Enter

Exercise:
- Enter marks for one class

---

### 4. Generating AI Remarks

Steps:
1. School Examination → Generate AI Remarks
2. Select class

Exercise:
- Generate remarks for a class

---

### 5. Recording Fees

Steps:
1. Accounting → Pay Fees
2. Select student
3. Enter amount
4. Press Enter

Exercise:
- Record 3 payments

---

### 6. Checking Reports

Steps:
1. Show Exams Report
2. Select class and student

Exercise:
- View a student report

---

### 7. Backup Training

Steps:
1. User Tools → Record Reservation
2. Select class
3. Click Reserve

Exercise:
- Backup one class

---

## Safety Training

⚠️ Important Rules:
- Always backup before major actions
- Do not delete students unless necessary
- Do not run promotions without confirmation
- Avoid closing the system during processing

---

## Common Mistakes to Avoid
- Entering wrong marks
- Selecting wrong class
- Skipping backups
- Misusing delete options

---

## Trainer Notes
- Allow users to practice after each section
- Repeat critical tasks (fees, exams, backup)
- Monitor first-time users closely

---

## Final Training Goal
By the end of training, users should be able to:
- Manage students
- Record fees
- Enter exam data
- Generate reports
- Use AI remarks confidently
