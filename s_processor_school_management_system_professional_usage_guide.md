S-Processor is a complete school management system that manages student data, finances, exams, and reporting with AI-powered insights.

---

## 2. Getting Started

### Installation
```bash
git clone https://github.com/AristotleRazak/s-processor-school-management-system.git
cd s-processor-school-management-system
```

### First-Time Setup
1. Open the application
2. Go to **User Tools → School Information**
3. Enter school details
4. Register students
5. Set term and fees

## 4. Student Management

### Register Student
1. User Tools → Register Student
2. Select class
3. Enter details
4. Click Register

### Delete Student
⚠️ WARNING: Permanent action
1. User Tools → Delete Student(s)
2. Select class
3. Select student(s)
4. Click Delete

### Edit Student Info
1. User Tools → Change Names and More
2. Select class
3. Modify details

---

## 5. Exams & Reporting

### Enter Marks (Per Class)
1. School Examination → Make School Exams → Per Class
2. Select class
3. Enter marks
4. Press Enter

### Enter Marks (Per Subject)
1. School Examination → Make School Exams → Per Subject
2. Select class and subject
3. Enter marks

### Clean Marks
- Temporary/Permanent: Enter **1000** as marks

### Generate Positions
1. School Examination → Generate Class Positions
2. Select class

### View Reports
1. Show Exams Report
2. Select class and student

---

## 6. AI Remarks

### Generate Automatically
1. School Examination → Generate AI Remarks
2. Select class

### Manual Correction
1. Navigate to remarks section
2. Click or Ctrl+Click remark
3. Edit and save

### Clear Remarks
- Enter empty value and save

---

## 7. Accounting & Fees

### Record Payment
1. Accounting → Pay Fees
2. Select class and student
3. Enter amount
4. Press Enter

### Track Payments
1. Collection Sheets
2. Filter by class/date

### Fees Status
- Fully Paid
- Not Fully Paid
- Not Paid

Path:
Accounting → Show Fees

---

## 8. Daily Collection Sheets

Generate sheets for:
- Canteen
- Bus
- School Fees
- Combined collections

Steps:
1. Daily Record → Select sheet type
2. Select class
3. Right-click → Print

### Editing Sheets
- Right-click → Edit rows/columns
- Change title
- Adjust layout

## 10. Backup & Recovery

### Backup Class
1. User Tools → Record Reservation
2. Select class
3. Click Reserve

### Restore Class
1. User Tools → Installations → Class Installation
2. Select file
3. Install

### Plugin Installation
1. User Tools → Installations → Plugin Installation
2. Select plugin
3. Install
